# 📊 kmer-counter

**K-mer频次统计与差异比较**

## 使用方法

```bash
cd kmer-counter
python kmer_counter.py
```

运行后按提示依次输入参数，所有参数均有默认值，直接回车即可使用默认值。

### 参数说明

| # | 参数 | 默认值 |
|---|------|--------|
| 1 | `FASTA文件路径` | `sequences.fasta` |
| 2 | `K值(4/5/6等)` | `5` |
| 3 | `对比FASTA路径(留空=无对比)` | `` |
| 4 | `展示TopN` | `20` |

### 交互式输入示例

```
FASTA文件路径 [默认: sequences.fasta]: 
K值(4/5/6等) [默认: 5]: 
对比FASTA路径(留空=无对比) [默认: ]: 
展示TopN [默认: 20]: 
```

## 依赖

无外部依赖，纯Python标准库

## 输出

脚本运行后会在当前目录生成结果文件，具体文件名见运行提示。
