# heatmap-annotation-builder

热图行列注释文件构建器

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| sample_file | 样本信息CSV(样本,组,批次等) | samples.csv |
| output_file | 输出注释文件路径 | heatmap_anno.csv |
| color_scheme | 配色方案(set1/set2/dark2) | set1 |

## 使用示例

```bash
python heatmap-annotation-builder.py
```

无额外依赖
