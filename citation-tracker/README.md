# 📖 citation-tracker

**论文被引追踪器**

## 使用方法

```bash
cd citation-tracker
python citation_tracker.py
```

运行后按提示依次输入参数，所有参数均有默认值，直接回车即可使用默认值。

### 参数说明

| # | 参数 | 默认值 |
|---|------|--------|
| 1 | `DOI或PMID(逗号分隔)` | `10.1038/nature12373` |
| 2 | `时间范围(近1年/3年/5年)` | `3y` |
| 3 | `最大返回数` | `100` |
| 4 | `出趋势图(yes/no)` | `yes` |

### 交互式输入示例

```
DOI或PMID(逗号分隔) [默认: 10.1038/nature12373]: 
时间范围(近1年/3年/5年) [默认: 3y]: 
最大返回数 [默认: 100]: 
出趋势图(yes/no) [默认: yes]: 
```

## 依赖

无外部依赖，纯Python标准库

## 输出

脚本运行后会在当前目录生成结果文件，具体文件名见运行提示。
