# 🎨 multi-panel-composer

**多子图组合排版**

## 使用方法

```bash
cd multi-panel-composer
python multi_panel_composer.py
```

运行后按提示依次输入参数，所有参数均有默认值，直接回车即可使用默认值。

### 参数说明

| # | 参数 | 默认值 |
|---|------|--------|
| 1 | `子图文件路径(逗号分隔)` | `fig1.png` |
| 2 | `布局(如2x2/3x1)` | `2x2` |
| 3 | `统一字体(yes/no)` | `yes` |
| 4 | `间距(0.0-0.2)` | `0.05` |
| 5 | `输出尺寸(如10x8)` | `10x8` |

### 交互式输入示例

```
子图文件路径(逗号分隔) [默认: fig1.png]: 
布局(如2x2/3x1) [默认: 2x2]: 
统一字体(yes/no) [默认: yes]: 
间距(0.0-0.2) [默认: 0.05]: 
输出尺寸(如10x8) [默认: 10x8]: 
```

## 依赖

无外部依赖，纯Python标准库

## 输出

脚本运行后会在当前目录生成结果文件，具体文件名见运行提示。
