# pairwise-distance-matrix

## 一句话说明
计算样本间距离矩阵(Euclidean/Correlation等)，用于聚类分析。

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 表达矩阵文件 | expression.tsv | 表达数据 |
| 输出距离矩阵 | distance_matrix.tsv | 距离矩阵 |
| 距离度量 | euclidean | euclidean/correlation |

## 使用示例

```bash
python pairwise_distance_matrix.py
```
