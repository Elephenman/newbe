# bioinformatics-pipeline-doc

生信分析流程文档自动生成器

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| input_file | 流程脚本路径(.py/.R/.sh) | pipeline.py |
| output_file | 输出文档路径 | pipeline_doc.md |
| title | 文档标题 | Analysis Pipeline |

## 使用示例

```bash
python bioinformatics-pipeline-doc.py
```

## 依赖

```
无额外依赖
```
