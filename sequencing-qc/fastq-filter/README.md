# 🧬 fastq-filter

**FASTQ按质量/长度/GC含量过滤**

## 使用方法

```bash
cd fastq-filter
python fastq_filter.py
```

运行后按提示依次输入参数，所有参数均有默认值，直接回车即可使用默认值。

### 参数说明

| # | 参数 | 默认值 |
|---|------|--------|
| 1 | `输入FASTQ文件路径` | `sample.fastq` |
| 2 | `最小平均质量阈值` | `20` |
| 3 | `最小序列长度(bp)` | `50` |
| 4 | `GC含量范围(如0.3-0.7)` | `0.3-0.7` |
| 5 | `输出文件路径(留空自动命名)` | `` |

### 交互式输入示例

```
输入FASTQ文件路径 [默认: sample.fastq]: 
最小平均质量阈值 [默认: 20]: 
最小序列长度(bp) [默认: 50]: 
GC含量范围(如0.3-0.7) [默认: 0.3-0.7]: 
输出文件路径(留空自动命名) [默认: ]: 
```

## 依赖

无外部依赖，纯Python标准库

## 输出

脚本运行后会在当前目录生成结果文件，具体文件名见运行提示。
