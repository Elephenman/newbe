# vcf-concordance-checker

比较两个VCF文件的变异一致性，输出concordance统计

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| vcf1 | 第一个VCF文件路径 | sample1.vcf |
| vcf2 | 第二个VCF文件路径 | sample2.vcf |
| output_file | 输出报告路径 | concordance.txt |

## 使用示例

```bash
python vcf-concordance-checker.py
```

## 依赖

```
无额外依赖
```
